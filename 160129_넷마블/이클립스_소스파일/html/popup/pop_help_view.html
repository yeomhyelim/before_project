계약관리

  • 1) 거래처
    • 퍼블리싱 계약의 상대 거래처를 입력한다. 팝업참을 통하여 입력 조회할 수 있고, 만일 거래처가 없는 경우
      거래처 등록을 먼저 진행해야합니다.
  • 2) 거래처 국가 : 거래처의 국가를 선택하여 입력합니다(팝업)
  • 3) 게임명 : 퍼블리싱 계약에 해당되는 게임명을 수기 입력 합니다.
  • 4) 장르 : 계임의 장르를 수기 입력합니다.
  • 5) 시장성 : 게임의 시장성을 수기 입력합니다.
  • 6) 특징 : 게임의 특징에 대해서 수기 입력합니다.
